# Userchap

:::tip API基础信息
- 接口功能：获取书籍ID对应的分享章节信息
- 请求方式：GET
- 频率限制：<span id="blue">10/10s*</span>
- 更新状态：<span id="green">已完工</span>

> *：此处频率限制为总频率限制
:::

<HTTP text={"GET /userchap/{pageid}"}/>

### 响应

以下示例显示了相应的响应。

```tsx title="Response"
[
    {
        "book_id": 100385826,
        "chapters": 16,
        "modes": "vip"
    },
    {
        "book_id": 100342791,
        "chapters": 194,
        "modes": "marauder vip"
    }
]
```

### 代码片段

可直接使用的TypeScirpt类型
```ts
export interface Userchap {
    book_id: number,
    chapters: number,
    modes: 'post vip' | 'marauder' | 'marauder vip' | 'vip'
}
```
