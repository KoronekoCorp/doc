# UserchapCount

:::tip API基础信息
- 接口功能：获取存在的书籍总数
- 请求方式：GET
- 频率限制：<span id="blue">10/10s*</span>
- 更新状态：<span id="green">已完工</span>

> *：此处频率限制为总频率限制
:::

<HTTP text={"GET /UserchapCount"}/>

### 响应

以下示例显示了相应的响应。

```tsx title="Response"
[
    {
        "COUNT(DISTINCT book_id)": 4265
    }
]
```
