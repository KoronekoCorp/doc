# Bookchaps

:::tip API基础信息
- 接口功能：获取书籍存在分享的章节列表
- 请求方式：GET
- 频率限制：<span id="blue">10/10s*</span>
- 更新状态：<span id="green">已完工</span>

> *：此处频率限制为总频率限制
:::

        <HTTP text={"GET /Bookchaps/{Bookid}"} />

### 响应

以下示例显示了相应的响应。

```tsx title="Response"
[
    {
        "chapter_id": 111003164,
        "mode": "vip"
    },
    {
        "chapter_id": 111003165,
        "mode": "vip"
    },
]
```

可直接使用的TypeScirpt类型

export interface Chaper {
