# isChapterPurchased

:::tip API基础信息
- 接口功能：判断对应章节ID是否存在分享
- 请求方式：GET
- 频率限制：<span id="blue">10/10s*</span>
- 更新状态：<span id="green">已完工</span>

> *：此处频率限制为总频率限制
:::

<HTTP text={"GET /isChapterPurchased/{chapid}"}/>

:::note
返回结果字符串，请自行判断
:::

### 响应

以下示例显示了相应的响应。

```tsx title="Response"
true
```

```tsx title="Response"
false
```
