# UserchapInfo

:::tip API基础信息
- 接口功能：获取书籍ID对应的分享章节摘要信息
- 请求方式：GET
- 频率限制：<span id="blue">10/10s*</span>
- 更新状态：<span id="green">已完工</span>

> *：此处频率限制为总频率限制
:::

        <HTTP text={"GET /UserchapInfo/{Bookid}"} />

### 响应

以下示例显示了相应的响应。

```tsx title="Response"
[
    {
        "chapters": 15,
        "modes": "vip"
    }
]
```

特殊说明：
mode包含vip、marauder

### 代码片段

可直接使用的TypeScirpt类型
```ts
export interface UserchapInfo {
    chapters: number
    modes: 'post vip' | 'marauder' | 'marauder vip' | 'vip' | null
}
```
