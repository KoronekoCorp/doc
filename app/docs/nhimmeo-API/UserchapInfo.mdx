# UserchapInfo

:::tip API基础信息
- 接口功能：获取书籍ID对应的分享章节摘要信息
- 请求方式：GET
- 频率限制：<span id="blue">10/s*</span>
- 更新状态：<span id="green">已完工</span>

> *：此处频率限制为总频率限制
:::

<HTTP text={"GET /nhimmeo/UserchapInfo/{Bookid}"}/>

```tsx title=Response
[
    {
        "chapters": number,
        "modes": string | null
    }
]
```
