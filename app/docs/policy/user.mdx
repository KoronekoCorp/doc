# 用户协议

:::tip
更新日期：2024/5/7
:::

## 欢迎您使用Nhimmeo相关服务！

您应当阅读并遵守《服务协议》（以下简称“本协议”），请您务必审慎阅读、充分理解各条款内容，特别是免除或者限制责任的条款，以及开通或使用某项服务的单独协议，并选择接受或不接受。限制、免责条款可能以加粗形式提示您注意。

除非您已阅读并接受本协议所有条款。您的下载、安装、使用、登录，调用API等行为即视为您已阅读并同意本协议的约束。

如果您未满18周岁，您不得出于任何目的使用本网站。

## 协议的范围

本规则是KoronekoCorp制定的关于获取和使用KoronekoCorp Nhimmeo相关服务的规则。本规则适用于KoronekoCorp提供的需要注册或下载功能的全部软件和服务，包括但不限于下述列表：

- n.koroneko.co
- \*nextjs.nhimmeo.ovh
- mnz.koroneko.co
- oauth.koroneko.co/api/object/

> \* 归属KoronekoCorp服务范畴

您使用KoronekoCorp的软件和服务时，须同时遵守各项服务的单独协议

## 免责申明

在服务中存在需要登录**第三方账户**的需求，关于第三方账户，我们重视对您财产的保护，我们使用各种技术和程序，以防信息的不当使用。但请您理解，由于技术的限制以及可能存在的问题，即便竭尽所能，也不可能始终保证百分之百的安全。

**您需要了解，您在使用本服务的过程中，有可能因我们可控范围外的因素而出现问题。我们不对此做任何的保证，用户需对使用自行承担风险**

## 限制

1. 禁止将批量注册KoronekoCorp账户
2. 禁止使用爬虫技术，例如Python等对相关服务造成过大压力
3. 其他违规行为
